About Me

Hi Saya Syahrul Abdulah, saya berasal dari Ciawi, Tasikmalaya - Jawa Barat, Indonesia. Saya lulusan SMK dan mempunyai hobi di bidang design grafis. Melalui hobi inilah saya berkecimpung di dunia kerja.

Saya adalah seorang freelancer design grafis. Sudah 2 tahun terakhir saya menekuni dunia freelancer. Saya menyediakan jasa pembuatan poster, logo dan desain web.

Saya terbiasa dengan deadline dan jam kerja malam. Tapi di hari minggu dan libur saya lebih suka untuk tidak bekerja. Software yang saya gunakan antara lain Adobe Photoshop, Adobe Illustrator, CorelDraw, dan Prezi

Di dunia freelancer saya mempunyai brand nama sebagai TropicalDust. Saya memakai nama ini sejak 1 tahun terakhir. Saya harap dengan nama ini kalian lebih mudah dalam menemukan saya di media sosial.
